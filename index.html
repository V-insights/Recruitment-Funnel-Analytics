Recruitment Funnel Analytics

Using Data Science to Understand and Improve Hiring Processes

Overview Dashboard Model Files

📌 Project Overview

This project analyzes a recruitment dataset to uncover insights across different hiring stages and predict candidate hiring outcomes using machine learning. The final result is an interactive dashboard that simplifies understanding of the recruitment funnel.

📊 Power BI Dashboard

The dashboard provides insights on:

  • Drop-off at each recruitment stage
  • Hiring trends by experience, education, gender
  • Predicted hiring outcomes

🤖 ML Model Summary

A Random Forest classifier was trained to predict hiring outcomes with an accuracy of 78.28% on the training data. Features included education level, major, experience, company size, and more.
